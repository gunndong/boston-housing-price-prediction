# [ML] 보스턴 주택 가격 결정 요인 분석 및 예측 모델 구축

## 다중 선형 회귀를 활용한 주요 변수 도출 및 예측 모델링

<aside>

Key Summary

- 주택 가격에 영향을 미치는 다양한 환경적/사회적 변수를 분석하고, 가격을 예측하는 머신러닝 베이스라인 모델 구축
- 데이터 탐색(EDA), 피어슨 상관관계 기반 주요 피처(Feature) 추출, 다중 선형 회귀(Multiple Linear Regression) 모델 설계 및 평가
- $R^2 Score$를 통해 모델의 설명력을 검증하고, 가격에 가장 큰 영향을 미치는 핵심 결정 요인(방의 개수, 하위 계층 비율)을 수치적으로 증명
- 기술 스택: Python, Pandas, Matplotlib,/Seaborn, Scikit-learn
</aside>

### 문제 정의

- 배경: 주택 가격은 단순한 면적뿐만 아니라 범죄율(crim), 환경 요인(nox, 일산화질소 농도), 접근성(dis) 등 수많은 외부 요인의 복합적인 영향을 받습니다.
- 분석 과제: 수집된 13개의 독립변수 중 **'어떤 요인이 집값에 가장 직관적이고 강력한 영향을 주는지'** 상관관계를 밝혀내고, 미지의 주택 가격을 예측할 수 있는 회귀 모델을 구축하고자 했습니다.

### 결과
Mean Squared Error (MSE): 7.630015927200283

Root Mean Squared Error (RMSE): 2.762248346401946

R-squared (R²): 0.8065329175674142
